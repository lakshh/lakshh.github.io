I attended the 5-Day AI Agents Intensive Course with Google. It was held online between 10th and 14th November. This course is most useful for software engineers, architects, and tech leads who want to explore and learn about Agents.

Each day had four parts:

  • A summary podcast on the topic for the day. This was interesting because it was AI-generated. It was a good example of dogfooding.
  • A whitepaper on the topic. Most papers were 50–70 pages.
  • A few code assignments on Kaggle notebooks. These showed the topic in action and made it easier to understand.
  • A livestream discussion led by experts from Google, NVIDIA, and Cohere.

The course covered the following topics:

  • Introduction to Agents
  • Tools and MCP
  • Context Engineering
  • Evaluation
  • Observability
  • Running Agents in Production
  • Agent2Agent (A2A) Protocol

There was also a Discord server for all participants. We used it to discuss the topics and help each other with the assignments. I took part in these forums and answered both technical and conceptual questions. This helped me understand GenAI and Agents at a deeper level. It was also good to meet curious minds and share ideas.

The code assignments use Google’s Agent Development Kit (ADK). They also use the gemini-2.5-flash-lite model. This model stays in the free tier. It will not cost anything if you want to try the assignments. The course concludes with an assignment to create an AI agents project showcasing a use case that leverages some of the key capabilities learned throughout this course.
